## Project Information

- **Project Name:** Whale Platform
- **Type:** Learning Management System (LMS)
- **Frontend:** Angular 17.3.5 with ng-zorro-antd
- **Backend:** .NET 9.0 ASP.NET Core
- **Database:** PostgreSQL 16
- **Containerization:** Docker
